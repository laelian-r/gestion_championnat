# Gestion Championnat

> Ce projet vise à gérer les championnats ainsi que les équipes et les classements de foot

## Pré-requis

1. Il faut installer Composer.

2. Il faut installer Symfony CLI.

## Installation

1. Exécutez les commandes :

   ```sh
    git clone {lien_du_repository_github}
   ```

   ```sh
   cd {nom_projet}
   ```

2. Entrez la commande suivante à la racine du projet :
   ```sh
   symfony serve
   ```
